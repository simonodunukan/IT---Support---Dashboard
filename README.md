# IT---Support---Dashboard
A cloud based IT support ticketing system built with Python and AWS

## Project Goals

- Build a functional IT support ticket managment system
- Deploy infrastructure using AWS clouf services
- Demonstrate understanding of IT Support workflow
- Practice full stack development skills

## Technologies Used

- **Frontend:** HTML, CSS, Javascript
- **Backend:** Python (Flask)
- **Database:** AWS RDS (PostgreSQL)
- **Hosting:** AWS (S3, CloudFront, or EC2)
- **Monitoring:** AWS CloudWatch
